# Not an easy problem!

<RefTeX>
  We typically cannot hope to calculate $\rmP(X \in A)$ explicitly.
  <Fragment style={{ display: 'block' }} index={ 0 }>
    <h3 style={{ color: 'var(--alt)' }}>Asymptotic approximations</h3>
    <div style={{ background: 'var(--altBg)', padding: '0.5em', border: '1px solid' }}>
      <Fragment style={{ display: 'block', position: 'relative' }} index={ 2 }>
        <strong style={{ color: 'var(--main)' }}>CLTs.</strong> 
        { 
          String.raw`
            families $(X^\epsilon)_{\epsilon > 0}$ have weak convergence $X^\epsilon_\ast\rmP \rightarrow \mu$.
          `
        }
        <Fragment>
          <Fragment only>
            <div style={{ position: 'absolute', right: '208px', bottom: '40px', color: 'red' }}>
              <div style={{ position: 'absolute', width: '48px', height: '37px', background: 'yellow', top: '5px', right: '13px' }} />
              <span style={{ position: 'absolute', width: '250px' }}>easier to evaluate <br /> (e.g. normal)</span>
            </div>
          </Fragment>
          { 
            String.raw`
              $$ \rmP(X \in A) = \rmP(X^\epsilon \in A) = X^\epsilon_\ast\rmP A  \approx \mu A $$
            ` 
          }
        </Fragment>
      </Fragment>
      <Fragment style={{ display: 'block', position: 'relative' }}>
        <strong style={{ color: 'var(--main)' }}>LDPs.</strong>
        { 
          String.raw`
            families $(X^\epsilon)_{\epsilon > 0}$ have systematic decay $\epsilon\log\rmP(X^\epsilon \in A) = -r(A) + o(1)$.
          ` 
        }
        <Fragment>
          <Fragment only>
            <div style={{ position: 'absolute', right: '247px', bottom: '45px', color: 'red' }}>
              <div style={{ 
                position: 'absolute', 
                width: '38px', 
                height: '25px', 
                background: 'yellow', 
                right: '25px', 
                top: '7px', 
              }} />
              <span style={{ position: 'absolute', width: '260px' }}>determined by variational problem over $A$.</span>
            </div>
          </Fragment>
          {
            String.raw`
              $$ \rmP(X \in A) = \rmP(X^\epsilon \in A) \approx e^{-r(A)/\epsilon} $$
            `
          }
        </Fragment>
      </Fragment>
    </div>
  </Fragment>
  <Fragment index={ 1 }>
    <h3 style={{ color: 'var(--alt)' }}>Numerical algorithms</h3>
    <div style={{ background: 'var(--altBg)', padding: '0.5em', border: '1px solid' }}>
      <Fragment style={{ display: 'block', marginBottom: '0.25em' }}>
        <strong style={{ color: 'var(--main)' }}>Monte Carlo.</strong> repeatedly sample from related measures.
      </Fragment>
      <Fragment style={{ display: 'block', marginBottom: '0.25em' }}>
        <strong style={{ color: 'var(--main)' }}>Transform inversion.</strong> approximate inverse of fourier/laplace transform.
      </Fragment>
      <Fragment style={{ display: 'block' }}>
        <strong style={{ color: 'var(--main)' }}>(Integro-)PDEs.</strong> Feynman-Kac, Kolmogorov, Fokker-Plank, etc.
      </Fragment>
    </div>
  </Fragment>
</RefTeX>

# Affine processes

<RefTeX>
  <blockquote>
    $X$ stochastically continuous time-homogeneous Markov with log-affine transform: 
    {
      String.raw`$$
        \rmE\Big(e^{\langle u, X_t \rangle} | X_0 = x \Big) = e^{\phi(t, u) + \langle\psi(t, u), x\rangle}
      $$`
    }
  </blockquote>
</RefTeX>

<RefTeX>
  <h3 style={{ color: 'var(--alt)', marginBottom: '0.5em' }}>Exhibit many properties.</h3>
  <Fragment>
    <h3 style={{ color: 'var(--alt)' }}></h3>
    <ul className="spaced">
      <Fragment><li>defined on nontrivial spaces, e.g. $\bbR^m_+\times\bbR^n$, $S_d^+$.</li></Fragment>
      <Fragment><li>parameterizations for such constraints.</li></Fragment>
      <Fragment><li>state-dependent drift and diffusion.</li></Fragment>
      <Fragment><li>state-dependent jumps.</li></Fragment>
      <Fragment><li>cross-exciting.</li></Fragment>
      <Fragment><li>finite and infinite jump activity.</li></Fragment>
      <Fragment><li>killing.</li></Fragment>
    </ul>
  </Fragment>
</RefTeX>
